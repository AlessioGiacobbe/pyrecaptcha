# Py-reCAPTCHA
a (not fully working) google recaptcha solver made in python.
requirements:
  - selenium 
  - urllib (to download things)
  - inflect
  - imagga api wrapper (https://github.com/meltwater/py-image-api-wrappers)
  - ImageMagick
